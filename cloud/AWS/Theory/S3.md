# S3 - Simple Storage Service

**FTP in the Cloud**

## Uses:

- Static Web sites
- Logs
- Backups
- etc.

## Core

- Buckets
    - Top-level container
    - Globally unique names
- Objects
    - Actual data/files
- Access Control
    - Bucket Policies (resource-based)
    - IAM Policies (identity-based)
    - Access Control List (legacy)
